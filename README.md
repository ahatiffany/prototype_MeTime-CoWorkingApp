# Booking App (prototype built with Lovable)
MeTime is an app for parents looking for short-term virtual offices with free childcare services that's fully vetted.

- Use Case

- Feature
- AI Co-Pilots

## Use Case
1. Solopreneur needs a short-term virtual office + childcare services while she meets clients.
2. Burnout stay-at-home dad of 3 needs short-term childcare services while he gets his hair trimmed.

## Persona
- Primary: Working Parents
- Seconday: Stay-at-Home Parents
- Tertiary: Regional Hotel Managers | Division Presidents

## Technology
- Tech 1: Location Tracking Services
  - Must Have - Android OS | Apple OS
- Tech 2: GPS Navigation
  - Must Have - Google Maps
  - Nice to Have - Apple Maps
- Tech 3: Calendar Booking
  - Must Have - Location | Date | Available Time | Duration (2 or 3 hours)
  - Nice to Have - Alternative Location (for the same Date, Time, and Duration)
- Tech 4: Federated Search for Hotel Amenities
  - Must Have - keyword search for co-working space options (WiFi, Coffeee Bar, Lounge, Quiet Zone, Conference)
  - Nice to Have - keyword search for personal care options (Spa Packages/Treatments, Hair Salon, Manicure/Pedicure, Waxing)

## Design Notes
1. AI Prompt 1: a mobile app that allows moms to book 2-3 hours of co-working space within a hotel chain in dense urban areas, e.g. Holiday Inn. Every booked appointment will also include free childcare services with vetted nannies and/or scheduled activities — similar to those offered at some gyms like 24 Hour Fitness.
2. AI Generated V1:
   - Hero section with clear value prop
   - Nearby location list (hotel-based co-working spaces)
   - Booking card with date/time selection
   - Childcare info with trust badges
   - Bottom mobile app navigation   
 3. AI Prompt 2: Add a company logo in the upper right corner titled “MeTime” with a subtitle “They play while you work.” Under each hotel listing, add other amenities like spa treatments or hair/nail services.
 4. AI Generated V2: (modified hero + new amenities)
 5. AI Visual Edits Prompt 1: Change transparency so How It Works is readable
